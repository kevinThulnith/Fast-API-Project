# Backend API

A simple FastAPI backend that serves fake data generated by [Faker](https://github.com/joke2k/faker). This project is designed to simulate a real-world API for testing and development purposes.

## Features

- **FastAPI**: High-performance web framework.
- **Faker**: Generates realistic fake data for users and posts.
- **Pydantic**: Data validation and settings management.
- **CORS Support**: Pre-configured for React applications running on `localhost` (ports 3000-3004).
- **Pre-generated Data**: Generates 500 users and their associated posts on startup.

## API Endpoints

### Users

- `GET /api/users`: Retrieve a list of all users.
- `GET /api/users/paginated`: Retrieve a paginated list of users.
- `GET /api/users/{user_id}`: Retrieve details for a specific user, including their posts.

### Posts

- `POST /api/posts`: Create a new post.
- `PUT /api/posts/{post_id}`: Update an existing post.
- `DELETE /api/posts/{post_id}`: Delete a post.

### Utilities & Testing

- `GET /api/slow-data`: Simulates a slow network response (configurable delay).
- `GET /api/large-payload`: Returns a large JSON payload for performance testing.
- `GET /api/metrics/db-size`: Returns the current number of users and posts in memory.

## Getting Started

### Prerequisites

- Python 3.14+
- [uv](https://github.com/astral-sh/uv) (recommended) or `pip`

### Installation

Clone the repository and install the dependencies:

```bash
# Using uv
uv sync

# Using pip
pip install -r requirements.txt
```

### Running the Application

Start the development server with:

```bash
# Using uv
uv run uvicorn main:app --reload --port 8000

# Using uvicorn directly
uvicorn main:app --reload
```

The API will be available at `http://localhost:8000`. You can access the interactive Swagger documentation at `http://localhost:8000/docs`.
